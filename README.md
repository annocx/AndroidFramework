# AndroidFramework
android 项目框架搭建，直接进入开发，集成Rxjava+ReTrofit+okHttp+MVP+glide， baseAdapter，baseActivity，权限管理等
对于新手来说，搭建一个android项目的框架，需要好几天的时间，本人把自己的框架开源出来，希望能够节省各位宝贵的时间更快的进入项目开发，也希望能够提出自己的问题
